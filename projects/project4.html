Colombian Electronic Music Map

Description

This is a WebGL program that displays an interactive solar system from both the birdseye view and a front view. You can move around using the w/a/s/d keys, zoom in and out with z/x keys, and rotate with q/e keys. The purpose of the project is to understand fundemental concepts of computer graphics which include model transformations and texture mapping.
